# TEST-1775759811 — Implement typed LRU cache module with TTL eviction and full test suite

Implement a production-quality LRU (Least-Recently-Used) cache module in TypeScript with TTL (time-to-live) support and a comprehensive Vitest test suite. The implementation uses a doubly-linked list combined with a Map for O(1) get and set operations, ensuring efficient cache performance with proper eviction and expiration handling.
